EasyAGAL by Barliesque

EasyAGAL is an open source ActionScript library that assists developers in writing AGAL ("Adobe Graphics Assembly Language") by providing an AS3-based pseudo-AGAL command set. The resulting advantages include: Code completion and hinting, more easy-to-read code, macros organized into libraries, and dynamic code customization. EasyAGAL provides training wheels to learning AGAL as well as a structural foundation for writing shaders for use with any other library. All methods of EasyAGAL maintain the same parameters, in the same order, as AGAL opcodes; They are all documented with ASDoc tags providing help for every instruction and register as you type, including the original AGAL opcodes.
