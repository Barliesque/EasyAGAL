color()method
public static function color(dest:IRegister, hslBaseColor:IRegister, hslBlendColor:IRegister):void

Color blend - The hue and saturation of the blend color are combined with the luminosity of the base color. Use ColorSpace.rgb2hsl() to convert RGB colors for use with this macro, then ColorSpace.hsl2rgb() to convert the result back to RGB.

Parameters

dest:IRegister — Register to store resulting HSL color.
 
hslBaseColor:IRegister — The HSL color of the pixel on top.
 
hslBlendColor:IRegister — The HSL color of the pixel underneath.
