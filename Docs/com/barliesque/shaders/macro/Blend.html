softLight()method 
public static function softLight(dest:IRegister, baseColor:IRegister, blendColor:IRegister):void

Soft-Light blending both darkens and lightens, depending on the blend color. If the blend color is lighter than 50%, the result is lightened as if it were dodged. If the blend color is darker than 50% gray, the image is darkened as if it were burned in.
