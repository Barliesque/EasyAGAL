posterize()method 
public static function posterize(dest:IRegister, source:IRegister, bands:IComponent, temp:IRegister):void

Reduces the range of color possibilities, turning smooth gradients into flat color bands. This can be used with either RGB or HSL color values, producing different effects. Note that this is a simplified form of posterization that does not base its output pallet on a selection of colors from the original image, like the posterize effect in Photoshop. [4 operations]

Parameters

dest:IRegister — Register to store resulting color
 
source:IRegister — Register containing the original color value
 
bands:IComponent — A component containing the total number of color bands (per channel)
 
temp:IRegister — A temporary register to be utilized for this calculation
