Interfacepublic interface IRegister extends IField, com.barliesque.agal.IIField

An AGAL register with four component values. Use this interface when assigning variables as "aliases" to registers, or when defining parameters for a macro function.

The fourth of four components contained in the register. May be accessed identically as either "a" (the alpha component in color notation), or "w" (for positional or vector notation). Color values range from 0.0 (darkest) to 1.0 (lightest), though the component may hold any numerical value.

Method Detail
_()method
public function _(xyzwrgba:String):IField

Specify any register component or components by string, for example: "zyx" "wwww" "gbr" The ability to specify an arbitrary arrangement of components is known as component "swizzling". Swizzling enables us, for example, to reduce the following series of instructions: mul( TEMP[0].x, TEMP[1].z, TEMP[1].w ); mul( TEMP[0].y, TEMP[1].x, TEMP[1].w ); mul( TEMP[0].z, TEMP[1].x, TEMP[1].y ); mul( TEMP[0].w, TEMP[1].z, TEMP[1].y ); Down to this single instruction: mul( TEMP[0]._("xyzw"), TEMP[1]._("zxxz"), TEMP[1]._("wwyy") );

Parameters

xyzwrgba:String — A string containing a maximum of four components
