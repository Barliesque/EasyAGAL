EasyAGALMethods
 
Packagecom.barliesque.agal
Classpublic class EasyAGAL
InheritanceEasyAGAL com.barliesque.agal.EasyBase

An easy-reading and easy-writing alternative to bare naked AGAL. Extend this class, or EasierAGAL, to write new shader programs. This class uses the same three-letter opcodes originally specified by AGAL. See the EasierAGAL class for a more verbose, easier-reading alternative.
